    Welcome to IntegraMOD3.

    IntegraMOD3 is a modded version of the latest stable release from phpBB, which is phpBB 3.0.6. This modded version already has preinstalled modifications available for the forum application. The term "IntegraMOD" means "integrated modifications" since the added modifications are customized to be compatible with each other. This version of IntegraMOD is more powerful and feature-packed with the integration of Stargate Portal which is a versatile and powerful homepage frontend for the phpBB forum. The version of Stargate Portal installed with IntegraMOD utilizes the forum-wide integration of the blocks so that all pages can display the portal blocks as desired, multi-page feature for additional portal pages like this, and a cache system for faster performance and lesser resources needed.
